# Convergence, 收斂的方法

* Split Horizon Update, 水平分割

 水平分割使指閘道器會紀錄所接收到的路徑資訊是由哪一個網路介面而來的，同時不可將此路徑傳回給同樣的網路介面，避免造成迴圈。
